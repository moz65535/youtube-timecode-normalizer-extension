## リンク単体の試験ページ

実際のリンク要素に対する右クリックメニューは、専用ページをローカル配信して確認します。

```powershell
npm run test:manual
```

このコマンドはFirefox版を`dist/firefox/`へ自動ビルドしてから試験ページを起動します。表示された `http://127.0.0.1:4173/` をChromeまたはFirefoxで開きます。ページ内には基本形式、パラメータ付き、崩れた形式、変換対象外のリンクと、初期設定での期待値を掲載しています。テスト終了時はターミナルで `Ctrl+C` を押して停止します。

同じページのtextareaでは、編集欄の選択範囲を直接書き換える処理、選択範囲外の保持、変更件数、Undoを確認できます。状態監視付きのcontenteditableでは、画面表示だけでなく`input`イベントを通じて保存対象の内部状態も更新されることを確認できます。

popup試験用の選択範囲には、主要カテゴリを網羅する11件のURLと、初期表示、フィルター、疑わしい対象外、コピー結果の期待件数を掲載しています。
タイムコード付きURLの`si`保持・除去は、基準件数へ影響しない専用の1件テストで確認できます。
`list`と`index`は、通常プレイリストの保持・除去と、`LL`・`WL`に付随する`index`の常時除去を専用ケースで確認できます。

ページ上部の「クリップボード確認用」は、任意のコピー結果を別窓へ移動せず確認するための汎用テキスト欄です。

- [ ] 変換前形式の維持、`www.youtube.com`、`youtu.be`の各形式を選択できる
