リンク単体テスト

各リンクを右クリックし、「リンクをタイムコード正規化URLにしてコピー」と 「正規化したリンクを開く」を確認します。

下表の期待値は初期設定（変換前形式を維持、si保持、feature除去、list非保持、 崩れた時刻を救済）を前提とします。

基本形式

済 ケース 右クリックするリンク コピーされる期待値 実際のコピー結果
通常形式 watch / 17m12s https://www.youtube.com/watch?v=qhH-azW3LJw&t=1032
短縮形式 youtu.be / 27m33s https://youtu.be/N3UkUjJ8UiY?t=1653
ライブアーカイブ live / 2765s https://www.youtube.com/watch?v=6mdcWcjeQcA&t=2765
Shorts shorts / 1m12s https://www.youtube.com/watch?v=qhH-azW3LJw&t=72
正規化済み watch / 10905 https://www.youtube.com/watch?v=OjVYtizH0ks&t=10905

パラメータと崩れた形式

済 ケース 右クリックするリンク コピーされる期待値 実際のコピー結果
si保持 si + 1m https://www.youtube.com/watch?v=qhH-azW3LJw&si=tracking&t=60
feature除去 feature + 1m https://www.youtube.com/watch?v=qhH-azW3LJw&t=60
通常list（初期設定では除去） 企画プレイリスト https://youtu.be/xI28vMLu7XQ?t=7332
個人用list（常に除去） list=LL https://www.youtube.com/watch?v=5vNh4QIR5Sc&t=468
崩れた時刻 youtu.be + &t https://youtu.be/PWS1ftGoeb8?t=1172

変更しないケース

済 ケース 右クリックするリンク 期待する動作
タイムコードなし YouTube動画 変換対象外として通知される
未対応の時刻 t=abc 変換対象外として通知される
YouTube以外 example.com 変換対象外として通知される
YouTubeの動画以外 チャンネルURL 変換対象外として通知される
